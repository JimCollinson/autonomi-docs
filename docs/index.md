This docs set covers the current merged developer interfaces for Autonomi: the daemon-based `ant-sdk` path and the direct Rust/CLI `ant-client` path.

## Where to start

**New to the toolchain?** Start with [What is Autonomi?](getting-started/introduction.md), then [Install antd](getting-started/install.md), then [Your First Upload](getting-started/hello-world.md).

**Ready to build?** Go to [Store and Retrieve Data](how-to-guides/store-and-retrieve-data.md) and [Handle Payments](how-to-guides/handle-payments.md).

**Need direct Rust or CLI access?** See [Using ant-client](getting-started/using-ant-client.md).

**Need reference material?** Use [REST API](sdk-reference/rest-api.md), [Daemon CLI: antd](sdk-reference/daemon-cli.md), and [Command Reference](cli-reference/command-reference.md).
